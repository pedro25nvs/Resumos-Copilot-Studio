# Resumos-Copilot-Studio
Informações sobre o conteúdo das aulas Copilot Studio  

Em principio não foi possivel me enquadrar em uma categoria que possibilitase-me a criação de uma conta gratuita no Microsoft 365

Pré-Requisitos:

Criar uma conta no Microsoft 365 (não foi possivel acessar o conteudo premium da conta);
Ter um computador. 

Operações : 
Criar um Copilot baseado em modelo (escolher um modelo de viagens ou "safe travels".
Criar um Copilot baseado em descrição com IA.
Criar um Copilot em Branco.
